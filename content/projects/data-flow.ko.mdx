## 프로젝트 개요

데이터 플로우는 비즈니스 데이터를 시각화하는 종합 분석 대시보드입니다. 실시간 모니터링과 자동화된 리포팅 기능을 제공합니다.

    - 사용자 정의 대시보드
    - 자동화된 리포트 생성
    - 데이터 내보내기 기능
    - 다중 사용자 협업

    - 데이터베이스: PostgreSQL, Redis
    - 배포: Docker, AWS
    - 시각화: D3.js
  </div>
</TwoColumn>

## 핵심 기능

### 1. 실시간 데이터 시각화

WebSocket 연결을 사용하여 실시간 차트를 구현했습니다. 페이지 새로고침 없이 데이터가 자동으로 업데이트됩니다.

<TwoColumn>
  <div>
    **사용자 경험**

    - 부드러운 애니메이션
    - 인터랙티브한 차트
    - 사용자 정의 시간 범위
    - 다양한 차트 유형
  </div>

  <div>
    **기술적 구현**

    - 실시간 업데이트용 WebSocket
    - 시각화용 D3.js
    - 캐싱용 Redis
    - 최적화된 쿼리 성능
  </div>
</TwoColumn>

### 2. 사용자 정의 대시보드

사용자가 위젯을 드래그 앤 드롭하여 개인화된 대시보드를 만들 수 있습니다. 레이아웃 지속성으로 사용자 정의가 저장됩니다.
