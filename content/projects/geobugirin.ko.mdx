## 프로젝트 개요

거북이린은 "신경 쓰지 않아도 되는" 자세 교정 데스크톱 애플리케이션입니다. 하루 종일 컴퓨터 앞에 앉아 있는 개발자들이 겪는 거북목 문제를, 사용자의 프라이버시를 완전히 존중하며 해결합니다.

## 핵심 기능

### 1. 프라이버시 우선 설계

거북이린은 사용자의 카메라 영상이나 이미지가 외부 서버로 전송되지 않도록 설계되었습니다. 모든 자세 판정과 분석이 사용자의 컴퓨터 내부에서 완료됩니다.

<TwoColumn>
  <div>

    - 모든 판정 로직 클라이언트 수행
    - 결과 집계 데이터만 선택적 전송
    - 사용자가 "감시당하지 않는" 구조
  </div>

  <div>
    **기술적 구현**

    - MediaPipe Pose로 로컬 추론
    - 기하학 수식 기반 자세 판정
    - 서버는 통계용 요약 데이터만 저장
    - WebSocket 없는 클라이언트 전용 처리
  </div>
</TwoColumn>

### 3. UX: 사용자의 집중을 끊지 않음

## 기술 스택

### Frontend
- **Electron**: 데스크톱 앱 프레임워크
- **React**: UI 컴포넌트
- **TypeScript**: 타입 안전성
- **Tailwind CSS**: 스타일링

### AI/ML
- **MediaPipe Pose**: 온디바이스 포즈 추정
- **간단한 기하학 수식**: 자세 판정

### Backend (optional)
- **Python**: 통계 데이터 처리
- **SQLite**: 로컬 데이터 저장
- **FastAPI**: 선택적 데이터 동기화
