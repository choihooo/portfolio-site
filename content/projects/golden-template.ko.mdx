## 핵심 기능

### 1. Atomic Design 구조

컴포넌트를 Atoms, Molecules, Organisms, Templates, Pages로 분리하여 재사용성을 극대화했습니다.

- **Atoms**: 버튼, 입력창, 타이포그래피 등 기본 요소
- **Molecules**: 검색 바, 카드 헤더 등 조합 요소
- **Organisms**: 헤더, 사이드바 등 복잡한 조합
- **Templates**: 페이지 레이아웃
- **Pages**: 실제 페이지

### 2. Forest 패턴

상태 관리 복잡성을 해결하기 위해 Forest 패턴을 적용했습니다. 부 컴포넌트에서 상태를 관리하고 자식 컴포넌트는 props만 받습니다.

## 기술 스택

### Frontend
- **Astro**: 메인 프레임워크
- **React**: 컴포넌트 작성

### Development

### Design System
- **Atomic Design**: 설계 원칙
- **Tailwind CSS**: 유틸리티 CSS

## 기술적 도전 과제

### 1. Astro 학습 곡선
팀원들에게 Astro와 아일랜드 아키텍처를 교육하는 시간이 필요했습니다.

### 2. 컴포넌트 설계 기준
어떤 컴포넌트를 Atom으로 분리하고 어떤 것을 Molecule로 합칠지에 대한 명확한 기준이 필요했습니다.

### 3. 마이그레이션 복잡성
기존 넥사크로 코드를 새로운 컴포넌트로 마이그레이션하는 작업이 예상보다 복잡했습니다.
