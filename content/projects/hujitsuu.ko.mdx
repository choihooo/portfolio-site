## 프로젝트 개요

![레거시 현대화](https://images.unsplash.com/photo-1517694712202-14dd9538aa97?w=1200&h=600&fit=crop)

넥사크로 기반 레거시 애플리케이션을 현대적인 컴포넌트 기반 아키텍처로 재설계하는 프로젝트입니다. 기존의 복잡한 렌더링 로직을 재사용 가능한 컴포넌트로 분리하여 유지보수성과 개발 생산성을 개선합니다.

## 핵심 기능

### 1. 컴포넌트 구조 설계

```
src/
├── components/           # 재사용 가능한 UI 컴포넌트
│   ├── ui/              # 기본 UI 컴포넌트 (Button, Input, etc.)
│   ├── business/        # 비즈니스 로직이 포함된 컴포넌트
│   └── layout/          # 레이아웃 컴포넌트
├── pages/               # 페이지별 컴포넌트
└── layouts/             # Astro 레이아웃
```

## 기술 스택

### Frontend
- **Astro**: 메인 프레임워크
- **React**: 컴포넌트 작성
- **TypeScript**: 타입 안전성

### Styling
- **CSS Modules**: 컴포넌트별 스코프드 스타일
- **Tailwind CSS**: 유틸리티 CSS 프레임워크

### Design System
- **Nexacro Theme**: 기업 표준 색상, 폰트 준수
- **일관된 Design System**: 간격, 색상, 타이포그래피

### 1. 넥사크로 그리드 시스템 이관
기존의 복잡한 그리드 정렬 로직을 React 컴포넌트로 어떻게 구현할지 고민이 필요합니다.

### 2. 기존 테마/스킨 시스템 마이그레이션
넥사크로의 기존 테마를 새로운 디자인 시스템과 어떻게 통합할지 결정해야 합니다.
