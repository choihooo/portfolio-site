## Project Overview

Modernization project for Korea University Software Technology Members Society (KUSITMS) official website using Astro and React. The project involves rebranding to the 33rd generation and includes code refactoring, design system construction, and GA integration.

## Key Features

<FeatureList icon="check">
  <FeatureItem>Optimal page load performance with Astro Island architecture</FeatureItem>
  <FeatureItem icon="palette">Complete UI/UX overhaul with 33rd generation branding</FeatureItem>
  <FeatureItem icon="chart">Google Analytics integration for visitor behavior tracking</FeatureItem>
  <FeatureItem icon="users">Onboarding and email collection for recruitment</FeatureItem>
  <FeatureItem icon="book">Activity record hub with blog recap section</FeatureItem>
</FeatureList>

### 2. Performance Optimization

<ProblemSolution>
  <Problem title="Image Optimization Issue">
    Exceeding Vercel's image optimization quota causes images to not display at all. The same issue occurred during the 31st Web TF.
  </Problem>

  <Solution title="Custom Image Optimization">
    Implemented custom image optimization instead of using Next.js Image tags. Selected one consistent image optimization method and documented the problem and solution in the portfolio.
  </Solution>
</ProblemSolution>

## Technical Challenges

<ProblemSolution>
  <Problem title="Island Architecture Design Complexity">
    Applying Astro's Island architecture while properly separating static content and client-side interactive components is challenging.
  </Problem>

  <Solution title="Clear Separation Layer Structure">
    - Static pages: SSR via Astro
    - Interactive components: React client-side load
    - Clearly separated project structure for maintainability
  </Solution>
</ProblemSolution>
