## プロジェクト概要

反復的な開発タスクを自動化するオープンソースコマンドラインインターフェースツールです。ワークフロー自動化により開発者の生産性を向上させます。

<TwoColumn>
  <div>
    ### 主な機能

    - プロジェクトスキャフォールディング
    - Gitワークフロー自動化
    - コード生成テンプレート
    - パッケージ管理ショートカット
    - クロスプラットフォーム対応
  </div>

  <div>
    ### 技術スタック

    - ランタイム: Node.js, TypeScript
    - CLIフレームワーク: Commander.js
    - UI: Ink, React
    - テスト: Jest
    - 配布: npm
  </div>
</TwoColumn>

## コア機能

### 1. プロジェクトスキャフォールディング

事前設定されたテンプレートで新しいプロジェクトを生成します。複数のフレームワークとベストプラクティスをすぐにサポートします。

<TwoColumn>
  <div>
    **ユーザー体験**

    - インタラクティブなCLIプロンプト
    - テンプレート選択
    - 依存関係インストール
    - Git初期化
  </div>

  <div>
    **技術的実装**

    - テンプレートエンジン
    - ファイルシステム操作
    - パッケージマネージャー統合
    - 設定可能なプリセット
  </div>
</TwoColumn>

### 2. Gitワークフロー自動化

コミットメッセージフォーマット、ブランチ管理、リリース公開などの一般的なGit操作を自動化します。
