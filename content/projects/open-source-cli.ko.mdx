## 프로젝트 개요

반복적인 개발 작업을 자동화하는 오픈 소스 명령줄 인터페이스 도구입니다. 워크플로우 자동화를 통해 개발자 생산성을 높여줍니다.

<TwoColumn>
  <div>
    ### 주요 특징

    - 프로젝트 스캐폴딩
    - Git 워크플로우 자동화
    - 코드 생성 템플릿
    - 패키지 관리 단축키
    - 크로스 플랫폼 지원
  </div>

  <div>
    ### 기술 스택

    - 런타임: Node.js, TypeScript
    - CLI 프레임워크: Commander.js
    - UI: Ink, React
    - 테스트: Jest
    - 배포: npm
  </div>
</TwoColumn>

## 핵심 기능

### 1. 프로젝트 스캐폴딩

미리 구성된 템플릿으로 새 프로젝트를 생성합니다. 다양한 프레임워크와 모범 사례를 즉시 지원합니다.

<TwoColumn>
  <div>
    **사용자 경험**

    - 인터랙티브 CLI 프롬프트
    - 템플릿 선택
    - 의존성 설치
    - Git 초기화
  </div>

  <div>
    **기술적 구현**

    - 템플릿 엔진
    - 파일 시스템 작업
    - 패키지 매니저 통합
    - 구성 가능한 프리셋
  </div>
</TwoColumn>

### 2. Git 워크플로우 자동화

커밋 메시지 포맷팅, 브랜치 관리, 릴리스 게시와 같은 일반적인 Git 작업을 자동화합니다.
