## 프로젝트 개요

Project Alpha는 원격 팀을 위한 실시간 협업 플랫폼입니다. WebSocket을 사용하여 실시간으로 동기화되는 작업 공간을 제공합니다.

<TwoColumn>
  <TwoColumnLeft>
    ### 주요 특징

    - 실시간 문서 공동 작성
    - 화면 공유 기능
    - 실시간 채팅
    - 세분화된 권한 관리
    - 반응형 디자인
  </TwoColumnLeft>

  <TwoColumnRight>
    ### 기술 스택

    - Frontend: React, TypeScript
    - Backend: Node.js, Express
    - Database: PostgreSQL
    - Cache: Redis
    - Real-time: WebSocket
  </TwoColumnRight>
</TwoColumn>

## 핵심 기능

### 1. 실시간 동기화

WebSocket 기반으로 초저지연성 실시간 동기화를 구현했습니다. 여러 사용자가 동시에 문서를 편집할 때 충돌 없이 자동으로 병합됩니다.

<TwoColumn>
  <div>
    **사용자 경험**

    - 자동 저장
    - 변경사항 실시간 반영
    - 오프라인 지원
  </div>

  <div>
    **기술적 구현**

    - Operational Transformation
    - Conflict-free Replicated Data Types
    - Optimistic UI
  </div>
</TwoColumn>

### 2. 화면 공유

WebRTC를 사용하여 P2P 화면 공유를 구현했습니다. 서버를 거치지 않고 직접 연결하여 지연을 최소화했습니다.
